# AI Arcade — Learn AI w/ Quiz

A retro arcade-style quiz game built with **Next.js 14** (App Router) for learning popular AI topics. Coffee-brown + pink CRT aesthetic.

## Topics (5 × 5 questions)

1. Large Language Models
2. Machine Learning Basics
3. Neural Networks
4. Computer Vision
5. AI Ethics

## Run

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Build

```bash
npm run build
npm start
```

## Stack

- Next.js 14 (App Router, TypeScript)
- Plain CSS with `Press Start 2P` + `VT323` Google Fonts for the 8-bit feel
- No external UI libraries
